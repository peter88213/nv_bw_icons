# User guide

This page refers to the latest 
[nv_bw_icons](https://github.com/peter88213/nv_bw_icons/) release.
You can open it with **Help > nv_bw_icons Online help**.

The plugin adds a **nv_bw_icons Online Help** entry to the **Help** menu. 

---

## How to restore the original icons

After uninstalling the *nv_bw_icons* plugin via the plugin manager, restart *novelibre*.
