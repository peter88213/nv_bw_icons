# Installation

If [novelibre](https://github.com/peter88213/novelibre/) is installed, the setup script *setup.py* installs the *nv_bw_icons* plugin in the *novelibre* plugin directory.


---

## How to restore the original icons

After uninstalling the *nv_bw_icons* plugin via the plugin manager, restart *novelibre*.
